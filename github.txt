1.git다운로드 설치(https://www.git-scm.com/)
  깃허브 가입(https://github.com ) /  repositories만들기

2. 내컴퓨터에 git을 설정할 폴더와 파일들을 만든다.(정한다)

3. git bash 열기 -> git을 설정할 폴더로 간다
	$ pwd (현재 위치 확인)
	$ cd .. (상위 디렉토리로 이동)
	$ cd ~ (최 상위, 루트로 이동)
	$ cd d: (d드라이브로 이동)
	$ ls (현재 위치해있는 디렉토리의 폴더와 파일을 보여줌)

4. $ git init (저장소로 만든다)
	$ ls -al (숨김파일까지 다 보임)
	$ git status (현재 상태 확인)
	$ git log (변경 이력 확인) 

5. github와 연결
	$ git config --global user.email "이메일주소"
	$ git config --global user.name "아이디"

6. 커밋지정(하나의 버전)
	$ git add 파일명 / $ git add . (추가하지 않은 모든 파일)
	$ git commit -m "메세지"

7.github사이트에서 토큰 받아오기
 	setting >  Developer settings > Personal access tokens > Generate new token버튼 클릭 > Note에 아무이름이나, repo체크-아래 Generate token버튼 클릭

8. github에 연결, 업로드
	git remote add origin https://github.com/아이디/저장소명.git
	git push -u origin master/main (파일 올리기, 재업로드시 에러 나면 +master)



== gitHub 호스팅으로 사이트 만들기 ==
저장소 setting > GitHub Pages > Source > none을 master로 > save버튼 클릭
